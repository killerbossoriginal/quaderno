| **PIL<br>Prodotto interno lordo**              | Somma di beni e servizi prodotti in un Paese in un anno.                                                                                                                                                                                                                                                | Oggi si usano anche altri indicatori di benessere che cercano di misurare la “felicità” dei cittadini.<br>L’assunto è che se in una società sempre più persone spendono soldi in medicine, armi, antifurti ecc. il PIL cresce, ma la cittadinanza non sarà necessariamente più felice.<br>Cfr. discorso di J.F. Kennedy, presidente americano |
| **Bilancia commerciale**                       | Il saldo tra importazioni e esportazioni di un Paese.                                                                                                                                                                                                                                                   |                                                                                                                                                                                                                                                                                                                                               |
| **Bilancio pubblico**                          | Il saldo tra le entrate e le uscite di un Paese.                                                                                                                                                                                                                                                        | C’è disavanzo (o **deficit**) se le uscite superano le entrate<br>C'è avanzo (o **surplus**) se le entrate superano le uscite.<br>-> La Destra storica puntò al pareggio di bilancio                                                                                                                                                          |
| **Debito pubblico**                            | Il totale del debito che ha un Paese nei confronti dei suoi creditori (privati cittadini, banche, altri Paesi...).                                                                                                                                                                                      |                                                                                                                                                                                                                                                                                                                                               |
| **Protezionismo**                              | Politica commerciale che mira a ostacolare le importazioni attraverso dazi doganali.                                                                                                                                                                                                                    |                                                                                                                                                                                                                                                                                                                                               |
| **Liberoscambismo<br>Liberismo**               | Politica commerciale che vuole l’abolizione dei dazi doganali e il libero commercio tra le nazioni.                                                                                                                                                                                                     |                                                                                                                                                                                                                                                                                                                                               |
| **Banche alla tedesca<br>Banche d'affari**     | Banche che si affermano a fine ‘800 che, oltre a raccogliere i risparmi e erogare prestiti, investono in attività imprenditoriali e commerciali.                                                                                                                                                        |                                                                                                                                                                                                                                                                                                                                               |
| **Monopolio**                                  | Situazione in cui un settore del mercato è controllato da un’unica azienda.                                                                                                                                                                                                                             |                                                                                                                                                                                                                                                                                                                                               |
| **Oligopolio**                                 | Situazione in cui un settore del mercato è controllato da poche aziende.                                                                                                                                                                                                                                |                                                                                                                                                                                                                                                                                                                                               |
| **Cartello<br>Trust**                          | Accordo tra aziende al fine di evitare di farsi concorrenza. I cartelli possono essere orizzontali (controllano la commercializzazione di un materiale, es. estrazione del petrolio) o verticali (controllano tutti i processi di lavorazione di un materiale dall’estrazione fino al prodotto finito). |                                                                                                                                                                                                                                                                                                                                               |
| **Holding**                                    | Società che controlla altre aziende attraverso partecipazioni o quote azionarie.                                                                                                                                                                                                                        |                                                                                                                                                                                                                                                                                                                                               |
